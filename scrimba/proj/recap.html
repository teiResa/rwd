Math Object

  • math.random() generates a pseudo-random number between 0 and 1 (not including 1, as in 1.0; 0.000 continued -> 0.999 continued)
  • math.random() * 6 (0.000 -> 5.999)
    • the multiplier is what dictates the highest end of the range.

  • let randoNum = Math.floor(Math.random() * 6) + 1
    return randoNum

    Returns a number 1 through 6. Without the +1, it would be 0 through 5. The floor makes it not a decimal. The *6 makes the .random number have a rande of 0.000 through 5.999.
  • math.floor() [positive numbers only here, but it works too] removes the decimal, so it is a whole number instead of a decimal

  • returns what it calls upon.
